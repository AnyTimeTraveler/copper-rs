Module uom::si::thermal_conductance

source ·
Expand description

Thermal conductance (base unit watt per kelvin, kg · m² · s⁻³ · K⁻¹).

Thermal conductance has the same kind as temperature interval, as this quantity relates to change of temperature. Not of kind TemperatureKind, used by thermodynamic temperature. See thermodynamic temperature for a full explanation.
