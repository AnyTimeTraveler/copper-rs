pub fn oneshot<T>() -> (OneshotSender<T>, OneshotReceiver<T>)
Expand description

Creates new oneshot channel and returns the OneshotSender<T> and OneshotReceiver<T> for it.

§Examples

 let (s, r) = kanal::oneshot();
 spawn(move || {
       s.send("Hello").unwrap();
      anyhow::Ok(())
 });
 let name=r.recv()?;
 println!("Hello {}!",name);
